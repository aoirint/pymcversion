# pymcversion

## Install

```shell
pip3 install git+https://github.com/aoirint/pymcversion.git
```

## Command Usage

```shell
mcversion bedrock

mcversion java

mcversion ios
```

## Develop

```shell
pip3 install -r requirements.txt
```

## Run tests

```shell
pip3 install -r requirements-test.txt

make test
```
